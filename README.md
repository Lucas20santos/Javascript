### Aula 03 - Tipo numerico e operações básicas

#### Operações

- Soma: 2 + 2 = 4

- Subtração: 2 - 2 = 0

- Multiplicação: 2 * 2 = 4

- Divisão: 2 / 2 = 1

- Resto: 2 % 2 = 0

- Retonar um valor inteiro: parseInt(11/2)

> "Wrapper Functions" nada mais sõa que funções construtoras. Ou seja, funções que constroem objetos.
